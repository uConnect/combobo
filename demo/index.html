Combobo

Combobo is an open-source, accessible combobox module. It can be used a single select, multi-select, or select-only dropdown without a text input box.

Single Select

Single select comboboxes are used when only one option is allowed to be selected at a time.

The <input> accepts text input and multiple text matching strategies ('starts-with', 'contains', 'equals') are available. These examples use different styles for matched text: one uses a background color and the other uses an underline style.

Multi-Select

Multi-select comboboxes are used when multiple options are allowed to be selected at a time. <input>-initialized comboboxes can enable this in their configs. <select multiple> will override the config settings.

These examples use different styles when multiple items are selected. The first shows { X selected } and the second concatenates the checked items with a dash separator.

    return selectedOptions.length > 1 ?
      '{ ' + selectedOptions.length + ' selected }' :
      selectedOptions[0].innerText.trim();

  optionValue: function (option) {
    const inputVal = box.multi.input.value;

Custom Toggle Icons

Custom toggle icons (such as text, <i class="fas fa-caret-down" />, or <svg>) can be passed in through the config.

For <input>-initialized comboboxes, the icon can also be added manually within the toggle instead. However, if an icon is also specified in the config, it will overwrite anything present inside the toggle button.

The demo page uses CSS to create a small triangle icon when the toggle button is empty.

Select-Only

Combobo can also be used as a select-only dropdown (without a text input box). This is useful when you'd like to use the same styling as your Combobo elements or to add more functionality than a traditional <select> allows. Though there is no visible text input, common keyboard behaviors (such as Arrow, Printable Characters, Enter, and Space behaviors from the W3C select-only combobox demo) are present.

The selectOnly setting works on comboboxes initiated from both <input> and <select> elements.

Initiated on <select>

Show Config 
{
  select: '#combobox-select-only',
  list: '.select-only .listbox',
  selectOnly: true,
  multiselect: true,
}
